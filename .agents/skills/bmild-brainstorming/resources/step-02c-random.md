## Random Selection

Embrace serendipity — random selection breaks habitual patterns. Load `./brain-methods.yaml` (keys: `category, technique_name, description`). Do not second-guess or substitute the random draw.

1. **Select** — Pick 3 techniques from at least 3 different categories. Ensure they don't actively conflict in approach (e.g., avoid three consecutive convergent analysis techniques). Otherwise: genuine random.

2. **Reveal** — Present with energy:

   ```
   **Random Technique Combination**

   **Phase 1 — [technique_name]**
   [Description — first 1–2 sentences from YAML]
   Unexpected angle: [What makes this technique surprising or unconventional]

   **Phase 2 — [technique_name]**
   [Description]
   How it plays with Phase 1: [Brief note on the unexpected interaction]

   **Phase 3 — [technique_name]**
   [Description]
   How it completes the journey: [Brief note]

   Total time: [sum of durations if applicable]
   ```

3. **Offer control** — Ask: *"Ready for this creative adventure? Or want a different roll of the dice?"* Options: [C] Start / [Shuffle] new random selection / [Details] explain a technique / [Back] return to approach selection.

4. **Hand off** — On confirmation, load `./resources/step-03-execute.md` carrying the selected technique names forward.
